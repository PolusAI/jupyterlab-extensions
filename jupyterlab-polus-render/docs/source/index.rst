
jupyterlab_polus_render
=====================================

Version: |release|

A Jupyterlab extension for RenderUI.


Quickstart
----------

To get started with jupyterlab_polus_render, install with pip::

    pip install jupyterlab_polus_render

or with conda::

    conda install jupyterlab_polus_render
