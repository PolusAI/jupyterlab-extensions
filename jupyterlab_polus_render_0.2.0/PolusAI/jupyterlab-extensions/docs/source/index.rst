
jupyterlab-polus-render
=====================================

Version: |release|

JupyterLab Polus Render makes Polus Render available as a JupyterLab extension.


Quickstart
----------

To get started with jupyterlab-polus-render, install with pip::

    pip install jupyterlab-polus-render

or with conda::

    conda install jupyterlab-polus-render
